# Button

The `Buttons` are a fundamental part of any project, with `Reactsax` you can add a great button with a single line of code.

<br />

## Default

The `Buttons` have all the states as `active`, `focus`, `hover`, and `disable` to make its implementation simpler and speed up the work.

<br />

#### Values :

- `active`
- `disabled`

## Colors

Change the color of the Button component with the `color` property and the color value (like `#fff`, `rgba (100,10,5)` or the main colors of `Reactsax`), this will only affect the component and its value can be dynamic. You can use color attribute or use `Reactsax` colors directly.

<br />

#### Color's Value :

- `primary`
- `success`
- `warning`
- `danger`
- `light`
- `dark`
- `RGB color` (like `rgb(255,255,25)`)
- `HEX color` (like `#3B3B98`)
